# ماژول WHMCS مدیریت کپچای کلودفلر(turnstile)

ماژول WHMCS رایگان و متن‌باز که جایگزین کپچای گوگل با [Cloudflare Turnstile](https://www.cloudflare.com/products/turnstile/) میشود. یک جایگزین کاربر محور.

## ویژگی‌ها

- **سازگاری کامل:** با قالب‌های پیشفرض (Six، Twenty-One) و همه قالب‌های سفارشی سازگار است.
- **داشبورد مدیریت:** مدیریت و شخصی سازی ویجت مستقیماً در داشبورد WHMCS.
- **کنترل صفحه:** فعال و غیرفعال‌سازی ویجت برای:
  - صفحه ورود
  - صفحه ثبت نام
  - بازنشانی رمز عبور
  - تماس با ما
  - ارسال تیکت پشتیبانی
  - سبد خرید
  - پرداخت
  - بررسی دامنه
- **شخصی سازی تم ویجت:** انتخاب بین تم خودکار، روشن یا تاریک.
- **انتخابگرهای پیشرفته:** تعریف انتخابگرهای CSS سفارشی برای تزریق ویجت به هر فرمی در هر قالبی بدون ویرایش فایل‌های قالب.
- **محدودسازی درخواست ها** برای جلوگیری از تعداد زیاد تلاش‌های ناموفق از یک آدرس آی پی
- **لیست سیاه و سفید آی پی** جهت دسترسی به صفحه بدون کپچا و قطع دسترسی کامل
- **پشتیبانی از متغییرهای های Smarty:** از `{display_turnstile}` در فایل‌های .tpl خود برای قرار دادن دستی استفاده کنید.

## نصب

1. آخرین نسخه را در [این صفحه](https://github.com/chaveamin/cloudgate/releases) دانلود کنید
2. نام پوشه را از `cloudgate-x.x.x` به `cloudgate` تغییر دهید.
3. پوشه cloudgate را در محل نصب WHMCS خود در آدرس `/modules/addons/` آپلود کنید:
4. وارد داشبورد ادمین WHMCS خود شوید.
5. به **پیکربندی > افزونه‌ها** بروید. کلودگیت را پیدا کنید و روی فعال‌سازی کلیک کنید.

## پیکربندی

**تنظیمات کلی**

| کلید         | مقدار پیشفرض | گزینه‌ها                  |
| ------------ | ------------ | ------------------------- |
| `site_key`   | خالی         | \*                        |
| `secret_key` | خالی         | \*                        |
| `theme`      | `auto`       | `auto` · `light` · `dark` |
| `size`       | `normal`     | `normal` · `compact`      |

**فعال‌سازی صفحات**

| صفحه         | کلید فعال‌سازی    | حالت پیشفرض |
| ------------ | ----------------- | ----------- |
| ورود         | `enable_login`    | غیرفعال     |
| ثبت‌نام      | `enable_register` | غیرفعال     |
| بازنشانی رمز | `enable_pwreset`  | غیرفعال     |
| ارتباط       | `enable_contact`  | غیرفعال     |
| تیکت         | `enable_ticket`   | غیرفعال     |
| سبد خرید     | `enable_cart`     | غیرفعال     |
| بررسی دامنه  | `enable_domain`   | غیرفعال     |

**تنظیمات**

| صفحه         | کلید            | مقدار پیشفرض | گزینه‌ها                                    |

| ورود         | `mode_login`    | `managed`    | `managed` · `non-interactive` · `invisible` |
| ثبت‌نام      | `mode_register` | `managed`    | `managed` · `non-interactive` · `invisible` |
| بازنشانی رمز | `mode_pwreset`  | `managed`    | `managed` · `non-interactive` · `invisible` |
| ارتباط       | `mode_contact`  | `managed`    | `managed` · `non-interactive` · `invisible` |

| بررسی دامنه  | `mode_domain`   | `managed`    | `managed` · `non-interactive` · `invisible` |

**محدودیت درخواست**

| کلید                 | مقدار پیشفرض | توضیح                     |
| -------------------- | ------------ | ------------------------- |
| `rate_limit_enabled` | غیرفعال      | فعال‌سازی محدودیت درخواست |
| `rate_limit_max`     | `5`          | حداکثر تلاش ناموفق مجاز   |
| `rate_limit_window`  | `5`          | بازه زمانی شمارش (دقیقه)  |

**لیست IP**

| کلید           | مقدار پیشفرض | توضیح                             |
| -------------- | ------------ | --------------------------------- |
| `ip_whitelist` | خالی         | آی پی معاف از کپچا                |

## نکته مهم

برای جلوگیری از تداخل، کپچای پیشفرض WHMCS را غیرفعال کنید:

1. به **پیکربندی > تنظیمات عمومی > امنیت** بروید.
2. **محافظ فرم ها با کپچا** را روی **همیشه غیرفعال** تنظیم کنید.

_توجه: این ماژول به طور خودکار سعی میکند عناصر کد امنیتی قدیمی را از طریق CSS پنهان کند، اما غیرفعال کردن آنها در تنظیمات توصیه میشود._

## استفاده دستی برای توسعه‌دهندگان

اگر ترجیح میدهید ویجت را به صورت دستی در فایل‌های قالب خود قرار دهید، میتوانید از متغییر Smarty استفاده کنید:

```html
<form method="post" action="login.php">
  {display_turnstile}
  <button type="submit">Login</button>
</form>
```
